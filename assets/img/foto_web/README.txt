ФОТО ДЛЯ САЙТА Skillful Hands Solutions — оптимизировано для web
================================================================

Что сделано
-----------
- Исходники из ..\foto (HEIC/JPG/PNG, 220 МБ) пережаты под web.
- Формат: JPG, длинная сторона <= 1920 px, quality 82, progressive,
  EXIF-ориентация выправлена, метаданные удалены.
- Итог: 106 файлов, ~48 МБ (в среднем ~450 КБ/фото).
- 3 дубля-копии («... (2)») отброшены.
- Имена — латиница, kebab-case, по проектам/категориям сайта,
  с состоянием before / during / after (до / в процессе / после).

Файлы соответствия:
  index.csv      — новое имя -> группа -> исходный файл
  _manifest.json — технический лог конвертации (размеры, разрешение)
  _contact_sheets\ — превью-листы со всеми фото (для быстрого просмотра)

Куда класть на сайте
--------------------
В коде (src/pages/project_detail.rs) предусмотрено:
  assets/img/projects/<slug>/...
slug-и: rustic-utility-room, waterfront-bbq, island-retreat.
Можно скопировать соответствующие группы в эти папки и подставить через
asset!("/assets/img/projects/<slug>/<file>") в hero_img / gallery.

Примечание: несколько кадров на стыке проектов (utility-room-after-07/08,
island-retreat-interior-before-08/09) отнесены по наиболее вероятному
признаку — при желании легко переименовать по index.csv.
